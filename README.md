### What it is?
JavaAVC (Java Audio/Video Codec) is FFmpeg wrapper for Java programming language. 

Currently supported FFmpeg 2.2.


### Requirements
 * Java 1.7 or newer.
 * *OS*: Linux or Windows, *Arch*: x86 or x86/64.
